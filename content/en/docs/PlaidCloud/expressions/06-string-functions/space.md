---
title: SPACE
---

Returns a string consisting of N blank space characters.

# Analyze Syntax

```python
func.space(<n>)
```

## Analyze Examples
```python
func.space(20)
+-----------------+
| func.space(20)  |
+-----------------+
|                 |
+-----------------+
```

## SQL Syntax

```sql
SPACE(<n>);
```

## Arguments

| Arguments | Description          |
|-----------|----------------------|
| `<n>`     | The number of spaces |

## Return Type

String data type value.

## SQL Examples

```sql
SELECT SPACE(20)
+----------------------+
| SPACE(20)            |
+----------------------+
|                      |
+----------------------+
```
