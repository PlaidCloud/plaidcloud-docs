---
title: OCT
---

Returns a string representation of the octal value of N.

# Analyze Syntax

```python
func.
```

## Analyze Examples
```python
func.
+-----------------+
| func. |
+-----------------+
|              50 |
+-----------------+
```

## SQL Syntax

```sql
OCT(<expr>)
```

## SQL Examples

```sql
SELECT OCT(12);
+---------+
| OCT(12) |
+---------+
| 014     |
+---------+
```
