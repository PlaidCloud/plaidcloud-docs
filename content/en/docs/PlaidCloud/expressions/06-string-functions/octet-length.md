---
title: OCTET_LENGTH
---

OCTET_LENGTH() is a synonym for LENGTH().

# Analyze Syntax

```python
func.
```

## Analyze Examples
```python
func.
+-----------------+
| func. |
+-----------------+
|              50 |
+-----------------+
```

## SQL Syntax

```sql
OCTET_LENGTH(<str>)
```

## SQL Examples

```sql
SELECT OCTET_LENGTH('databend');
+--------------------------+
| OCTET_LENGTH('databend') |
+--------------------------+
|                        8 |
+--------------------------+
```
