---
title: BIN
---

Returns a string representation of the binary value of N.

## SQL Syntax

```sql
BIN(<expr>)
```

## Arguments

| Arguments | Description |
|-----------|-------------|
| `<expr>`  | The number. |

## Return Type

`VARCHAR`

## SQL Examples

```sql
SELECT BIN(12);
+---------+
| BIN(12) |
+---------+
| 1100    |
+---------+
```
