---
title: COT
---

Returns the cotangent of `x`, where `x` is given in radians.

## Analyze Syntax

```python
func.cot( <x> )
```

## Analyze Examples

```python
func.cot(12) 

┌─────────────────────┐
│    func.cot(12)     │
├─────────────────────┤
│ -1.5726734063976895 │
└─────────────────────┘
```

## SQL Syntax

```sql
COT( <x> )
```

## SQL Examples

```sql
SELECT COT(12);

┌─────────────────────┐
│       cot(12)       │
├─────────────────────┤
│ -1.5726734063976895 │
└─────────────────────┘
```
