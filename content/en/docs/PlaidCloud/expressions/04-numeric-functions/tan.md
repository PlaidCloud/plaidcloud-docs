---
title: TAN
---

Returns the tangent of `x`, where `x` is given in radians.

## Analyze Syntax

```python
func.tan( <x> )
```

## Analyze Examples

```python
func.tan(90)

┌────────────────────┐
│    func.tan(90)    │
├────────────────────┤
│ -1.995200412208242 │
└────────────────────┘
```

## SQL Syntax

```sql
TAN( <x> )
```

## SQL Examples

```sql
SELECT TAN(90);

┌────────────────────┐
│       tan(90)      │
├────────────────────┤
│ -1.995200412208242 │
└────────────────────┘
```
