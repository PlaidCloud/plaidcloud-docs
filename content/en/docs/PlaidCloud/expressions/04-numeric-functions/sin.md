---
title: SIN
---

Returns the sine of `x`, where `x` is given in radians.

## Analyze Syntax

```python
func.sin( <x> )
```

## Analyze Examples

```python
func.sin(90)

┌────────────────────┐
│    func.sin(90)    │
├────────────────────┤
│ 0.8939966636005579 │
└────────────────────┘
```

## SQL Syntax

```sql
SIN( <x> )
```

## SQL Examples

```sql
SELECT SIN(90);

┌────────────────────┐
│       sin(90)      │
├────────────────────┤
│ 0.8939966636005579 │
└────────────────────┘
```
