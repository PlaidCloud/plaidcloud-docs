---
title: RADIANS
---

Returns the argument `x`, converted from degrees to radians.

## Analyze Syntax

```python
func.radians( <x> )
```

## Analyze Examples

```python
func.radians(90)

┌────────────────────┐
│  func.radians(90)  │
├────────────────────┤
│ 1.5707963267948966 │
└────────────────────┘
```

## SQL Syntax

```sql
RADIANS( <x> )
```

## SQL Examples

```sql
SELECT RADIANS(90);

┌────────────────────┐
│     radians(90)    │
├────────────────────┤
│ 1.5707963267948966 │
└────────────────────┘
```
