---
title: RADIANS
---

Returns the argument `x`, converted from degrees to radians.

## SQL Syntax

```sql
RADIANS( <x> )
```

## SQL Examples

```sql
SELECT RADIANS(90);

┌────────────────────┐
│     radians(90)    │
├────────────────────┤
│ 1.5707963267948966 │
└────────────────────┘
```
