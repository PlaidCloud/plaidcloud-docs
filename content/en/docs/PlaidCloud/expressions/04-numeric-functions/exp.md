---
title: EXP
---

Returns the value of e (the base of natural logarithms) raised to the power of `x`.

## SQL Syntax

```sql
EXP( <x> )
```

## SQL Examples

```sql
SELECT EXP(2);

┌──────────────────┐
│      exp(2)      │
├──────────────────┤
│ 7.38905609893065 │
└──────────────────┘
```
