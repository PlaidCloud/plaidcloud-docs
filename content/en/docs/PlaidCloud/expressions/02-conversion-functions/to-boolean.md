---
title: TO_BOOLEAN
---

Converts a value to BOOLEAN data type.

## SQL Syntax

```sql
TO_BOOLEAN( <expr> )
```

## SQL Examples

```sql
SELECT TO_BOOLEAN('true');

┌────────────────────┐
│ to_boolean('true') │
├────────────────────┤
│ true               │
└────────────────────┘
```
